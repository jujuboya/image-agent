## Phase 1: 环境搭建与基础测试 (1-2天)

### 1.1 环境准备
- [ ] 创建 `.env` 配置文件
- [ ] 安装 Python 依赖: `pip install -r requirements.txt`
- [ ] 安装 Node.js 依赖: `cd frontend && npm install`
- [ ] 配置 MySQL 数据库
- [ ] 配置 Redis (可选)
- [ ] 配置 RabbitMQ (可选)

### 1.2 数据库初始化
```bash
# 执行 SQL 初始化脚本
mysql -u root -p < sql/init.sql
```

### 1.3 后端服务启动测试
```bash
cd backend
python main.py
# 验证: http://localhost:8000/docs
```

### 1.4 前端服务启动测试
```bash
cd frontend
npm run dev
# 验证: http://localhost:5173
```
